Installation

Clone the repository:

git clone https://github.com/tarak-rowthu/Portfolio.git

Navigate to project directory:

cd Portfolio

Install dependencies:

npm install

Run development server:

npm run dev

Build for production:

npm run build

Frontend deployed using Vercel.
